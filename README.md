# Eight Queens

Eight Queens is it a learning exercise that solves a eight queens chess puzzle.

The eight queens puzzle is the problem of placing eight chess queens on an 8Ã—8 chessboard so that no two queens threaten each other; thus, a solution requires that no two queens share the same row, column, or diagonal. There are 92 solutions.(Wikipedia)

## Usage

Click a number in the left menu to represent one of the 92 solutions.
